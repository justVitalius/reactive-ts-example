# Reactive Typescript Example App

## Условия задачи
На rx.js написать программу, которая
двигает фигурку по полю слева направо.
На поле есть фишки разной ценности.
Сделав N ходов и дойдя до края нужно вывести
- кол-во фишек
- кол-во шагов
